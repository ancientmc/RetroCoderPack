== JarSplitter ==
Use: Splitting Minecraft's JAR file into two components: a file containing the sources, and a file containing resource files (textures, etc.)
Developers: Neoforged
Repository: https://github.com/neoforged/InstallerTools
License: GNU Lesser General Public License v2.1, https://github.com/neoforged/InstallerTools/blob/master/LICENSE.txt

== AutoRenamingTool ==
Use: Deobfuscating Minecraft's source code.
Developers: Neoforged
Repository: https://github.com/neoforged/AutoRenamingTool
License: GNU Lesser General Public License v2.1, https://github.com/neoforged/AutoRenamingTool/blob/master/LICENSE.txt

== MCInjector ==
Use: Injecting exceptions and other data into Minecraft's source code to ensure it works properly.
Developers: Mod Coder Pack Team (original), AncientMC (fork)
Original Repository: https://github.com/ModCoderPack/MCInjector
Forked Repository: https://github.com/ancientmc/MCInjector

== Fernflower ==
Use: Decompiling Minecraft's source code.
Developers: JetBrains (original), Minecraft Forge & RetroGradle (patches), AncientMC (fork)
Original repository: https://github.com/MinecraftForge/FernFlower, split from https://github.com/JetBrains/intellij-community/tree/master/plugins/java-decompiler/engine
Patch repository: https://github.com/MinecraftForge/ForgeFlower, https://github.com/RetroGradle/ForgeFlower
Forked repository: https://github.com/ancientmc/FernFlower

== DiffPatch ==
Use: Patching out any errors in the decompiled Minecraft code.
Developers: ChickenBones (original), TheCBProject (Java port), Minecraft Forge (fork)
Original repository: https://github.com/Chicken-Bones/DiffPatch
Java port repository: https://github.com/TheCBProject/DiffPatch
Forked repository: https://github.com/neoforged/DiffPatch
License: MIT License, https://github.com/neoforged/DiffPatch/blob/master/LICENSE.txt
